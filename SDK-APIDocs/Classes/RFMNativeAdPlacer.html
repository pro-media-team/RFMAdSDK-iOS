Inherits from NSObject
Declared in RFMNativeAdPlacer.h

Overview

RFMNativeAdPlacer class allows you to request native ads from the RFM ad server to be placed in your table view.

After creating an instance of RFMNativeAdPlacer, make a call to request native ads. You may then request a native ad response for a given index path.
