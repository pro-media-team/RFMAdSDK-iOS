RFMFastLane Class Reference

Inherits from NSObject
Declared in RFMFastLane.h

Overview

RFMFastLane class that handles fastlane prefetching and callbacks.

After creating an instance of RFMFastLane, make a call to prefetch an ad. It is recommended to move the primary SDK request call into the didReceiveFastLaneAdInfo: method.
