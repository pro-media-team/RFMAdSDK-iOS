RFMVideoAdSession Class Reference

Inherits from RFMAdSession : NSObject
Declared in RFMVideoAdSession.h

Overview

This class manages video playback session state and serves as a container to store opaque video ad instances. In addition, it exposes methods to start the session, pause video ad playback, resume video ad playback, and skip video ad playback.
