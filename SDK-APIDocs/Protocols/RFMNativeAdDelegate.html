RFMNativeAdDelegate Protocol Reference

Conforms to NSObject
Declared in RFMNativeAd.h

Overview

Native ad protocol for receiving native ad callbacks and notifications.

The native ad delegate should conform to this protocol.

– didRequestNativeAd:withUrl:

Optional Delegate callback when native ad request has been sent to server.

- (void)didRequestNativeAd:(RFMNativeAd *)nativeAd withUrl:(NSString *)requestUrlString

Parameters

nativeAd

The instance of RFMNativeAd for which this callback has been triggered.
requestUrlString

The request URL for the request sent to RFM ad server.

Discussion

This callback is triggered when a native ad request has been successfully sent to the ad server. Please note that this does not signify that a response has been received from the ad server. This callback is useful for checking the request URL that is actually sent to the ad server.
