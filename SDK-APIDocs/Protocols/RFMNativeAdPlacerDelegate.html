RFMNativeAdPlacerDelegate Protocol Reference

Conforms to NSObject
Declared in RFMNativeAdPlacer.h

Overview

Native ad placer protocol for receiving native ad callbacks or notifications using native ad placer.

The native ad placer delegate should conform to this protocol.

– didRequestNativeAdForIndexPath:placer: required method

Optional Delegate callback when native ad request using native ad placer has been sent to server.

- (void)didRequestNativeAdForIndexPath:(NSIndexPath *)indexPath placer:(RFMNativeAdPlacer *)placer

Parameters

indexPath

The cell index path for which this callback has been triggered.
placer

The native ad placer for which this callback has been triggered.

Discussion

This callback is triggered when a native ad request using native ad placer has been successfully sent to the ad server. Please note that this does not signify that a response has been received from the ad server.
